XMLPULL downloads

There are following file available for XMLPULL V1 API:

  • src: this contains only API source and build scripts, available as tarball-src (*src.tgz) and zip-src (*src.zip) files
  • all: contains all what is in src and precompiled .class, generated and ANT and JUnit .jar files - basically all needed to regenerate XMLPULL V1 API distribution
  • jar: pre-compiled .JAR files with API and TESTS. if you already have XMLPULL implementation API jar file should have been included with it and you should not need to download them however you may want to dwownload jar file with tests to verify if implementation is conformaing to XMLPULL V1 API:
    • xmlpull_*.jar: contains only API
    • xmlpull-tests_*.jar: contains JUnit tests fro API
  • unpacked contains directory with unpacked all-tarball so you can browse its content before downloading
